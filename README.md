# 事前準備
適当な場所に以下のプロジェクトをクローンしましょう

```
git clone https://github.com/georgesekkie/exam_mooovi.git
```

ディレクトリを移動して以下のコマンド順々に打ちましょう。

```
$ rake db:create
$ rake db:migrate
```
